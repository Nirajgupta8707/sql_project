
# Flight Operations Analytics | SQL Project

## Project Overview
This project analyzes airport and airline operations data to identify passenger traffic trends, route performance, flight frequency, and seat utilization. The analysis helps understand airport performance, high-traffic routes, and airline capacity utilization.

## Objectives
- Analyze passenger traffic across routes
- Identify busiest and least busy routes
- Evaluate average passengers per flight
- Identify high-traffic corridors
- Analyze seat utilization for airlines
- Identify popular destination airports
- Study relationship between city traffic and airport activity

## Tools Used
- SQL
- MySQL
- Aggregations
- Joins
- CTE (Common Table Expressions)
- Group By
- Data Analysis

## Key Analysis Performed
1. Passenger Traffic Analysis
2. Route Performance Analysis
3. Flight Frequency Analysis
4. Airport Performance Analysis
5. Seat Utilization Analysis
6. Destination Popularity Analysis
7. City vs Passenger Traffic Analysis

## Business Insights
- Identified busiest flight routes with highest passenger traffic.
- Found high-traffic corridors between major cities.
- Identified top-performing airports based on passenger volume.
- Calculated airline seat utilization efficiency.
- Identified most popular destination airports.
- Provided insights for route optimization and capacity planning.
